# rent-a-car

## Description
```
This car rental application was developed as part of a team project in the field of "Human-Computer Interaction". 
The app is built using React JS and currently only has a frontend. Although it is not completely finished 
at the moment, it represents a good basis for further development and upgrading.
3 team members participated in the development of the application.
```

## Functionalities
```
The goal of the application was to make it easier for users to interact, to reach their goal as easily 
as possible, in this case it is renting a car. Also, the vision was to improve the application 
compared to standard applications of that type.
```

## Installation

```
npm install -> to install dependencies
npm start -> for run app
```
